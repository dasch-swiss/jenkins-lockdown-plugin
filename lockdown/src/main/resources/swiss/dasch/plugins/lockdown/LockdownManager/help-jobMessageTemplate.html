Template used to render the job lockdown message. This template is rendered for each locked down job. The following patterns can be used in the template:

  • Display name of the job: {%name%}
  • Full name of the job: {%fullname%}
  • Description of the job: {%description%}
  • Absolute URL of the job: {%url%}
  • Lockdown reason: {%reason%}
  • Name of the user that started the lockdown: {%username%}
  • ID of the user that started the lockdown: {%userid%}
