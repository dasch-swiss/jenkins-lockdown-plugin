Whether the Jenkins system message should be used to display the lockdown message. If set to false, the lockdown message won't be shown in the classic Jenkins UI.
However, it remains accessible through the /jenkins/lockdown/lockdownMessage URL or, e.g., in Blue Ocean if the Lockdown (Blue Ocean) plugin is installed.
The system message will only be overwritten if it is empty or has been set by the Lockdown plugin. Custom system messages will never be overwritten.
